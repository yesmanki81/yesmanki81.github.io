🤖 Used Car Price Prediction in Virginia

This project demonstrates a full-stack machine learning pipeline built to predict used car prices in Virginia, covering the entire AI lifecycle — from data acquisition to model deployment.

📊 Project Overview

Using a real-world dataset containing over 46,000 used car listings, the system is built on an AWS EC2 instance with the following key components:

  • Data Collection & Ingestion

    • Data was crawled and stored in an AWS-hosted MySQL database
    • Loaded via Python and served using Flask

  • Feature Engineering & Preprocessing

    • Structured features include: mileage, model, year, transmission, fuel type, etc.
    • Handled missing data, transformed categorical variables, and performed standardization

  • Exploratory Data Analysis (EDA)

    • Visualized distributions and relationships using Pandas and Matplotlib
    • Correlation analysis to reduce redundancy and improve interpretability

🧠 Model Development & Evaluation

A series of regression models were trained and evaluated:

  • Linear Regression
  • Ridge Regression
  • Lasso Regression
  • Random Forest Regressor
  • XGBoost Regressor

Evaluation metrics:

  • Mean Absolute Error (MAE)
  • Root Mean Squared Error (RMSE)
  • R² Score

✅ XGBoost Regressor was chosen as the final model due to its superior performance across all metrics and its ability to generalize well to unseen data.

🚀 Deployment

  • The trained model was serialized using Pickle
  • Served via a lightweight Flask API
  • Ready for real-time inference with unseen car listings

🧩 Technologies Used

Layer Tech
Data Source AWS MySQL
Backend Python, Flask
Modeling scikit-learn, XGBoost
Deployment Pickle, REST API
Platform AWS EC2 (Ubuntu 22.04)

🏁 Final Result

This project showcases my ability to execute the complete machine learning pipeline, incorporating elements of:

  • Data Engineering
  • Supervised Learning
  • Model Selection & Tuning
  • API Deployment
  • Lightweight MLOps

It reflects production-level skill in building scalable, real-world AI solutions — particularly relevant for applied AI and ML engineering roles.

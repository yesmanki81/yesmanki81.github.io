🤖 Used Car Price Prediction in Virginia

A full-stack machine learning project that demonstrates the end-to-end AI workflow — from data acquisition to model deployment.

Leveraging a dataset of over 46,000 used car listings across Virginia, the project involves:

  • Automated data ingestion from AWS-hosted MySQL using Python and Flask
  • Feature engineering on structured automotive data (e.g., mileage, brand, transmission, year)
  • EDA and correlation-based feature selection
  • Model benchmarking using:
    • Linear Regression
    • Ridge Regression
    • Lasso Regression
    • Random Forest Regressor
    • XGBoost Regressor
  • XGBoost Regressor was selected based on MAE, RMSE, and R² performance
  • Model deployment via Flask API using Pickle serialization
This project reflects strong capabilities in data engineering, ML modeling, and lightweight MLOps, and showcases my ability to deliver production-grade AI pipelines as an AI Engineer.

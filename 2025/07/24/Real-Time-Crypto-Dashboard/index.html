Real-Time-Crypto-Dashboard

Created2025-07-24|Updated2025-07-25|Showcase
|Post Views:

This project builds a real-time cryptocurrency dashboard for Bitcoin and Ethereum using fully open-source tools on an AWS EC2 instance. The system collects live market data, streams it, stores it in a database, applies price prediction, and visualizes everything through Grafana.

---

⚙️ Architecture

🚀 Stack Overview

  • Data Source: CryptoCompare API (via requests)
  • Streaming: Apache Kafka
  • Storage: PostgreSQL
  • Prediction: Python (Simple Moving Average)
  • Visualization: Grafana
  • Environment: Ubuntu 22.04 on AWS EC2

⚡ Quick Setup Guide

1️⃣ Kafka

2️⃣ PostgreSQL

3️⃣ Python Scripts

  • crypto_producer.py: Sends BTC/ETH price to Kafka every 5 seconds
  • crypto_consumer.py: Reads from Kafka and inserts into PostgreSQL
  • price_predictor.py: Calculates 5-minute SMA and stores predictions

📈 Grafana Setup

  • Install: sudo apt install grafana
  • Access: http://<EC2_IP>:3000 (default credentials: admin / admin)
  • Add PostgreSQL as data source
  • Create dashboards:
    • Time series for real-time price
    • Line chart overlay for predictions
    • Stat panel for 24h high/low
    • Bar chart for 24h % change

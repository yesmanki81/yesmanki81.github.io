Real-Time-Crypto-Dashboard

Created2025-07-24|Updated2025-07-25|Showcase
|Post Views:

This project builds a real-time cryptocurrency dashboard for Bitcoin and Ethereum using fully open-source tools on an AWS EC2 instance. The system collects live market data, streams it, stores it in a database, applies price prediction, and visualizes everything through Grafana.
